# 飲食店 社員専用ポータル — 画面・機能 提案一覧

最初に整理した項目です。ダッシュボードに追加する候補を選ぶときの参考にしてください。

---

## 1. 共通

- ログイン / ログアウト
- 権限エラー画面（権限のないURLにアクセスしたとき）

---

## 2. ダッシュボード（役割別の想定）

### 一般社員向けダッシュボード

| 項目 | 内容 |
|------|------|
| 今日のシフト | 勤務時間・担当ポジション |
| 今日の売上目標・進捗 | ざっくりしたバー表示など |
| 重要なお知らせ | 本部・店長からのお知らせを数件ピックアップ |
| 自分あてのタスク | 例：清掃チェック、研修動画の視聴など |

### 店長・マネージャー向けダッシュボード

| 項目 | 内容 |
|------|------|
| 本日・今週の売上速報 | 目標比など |
| 予約状況 | 件数、人数、ピーク時間帯の混雑度 |
| 今日の出勤予定者 | 遅刻・欠勤のアラート |
| 未対応タスク | 勤怠修正申請、シフト変更申請、クレーム対応など |
| スタッフ日報の要約 | 日/週/月単位のAI要約（※実装済み） |
| 店長・マネージャー日報（同一期間） | （※実装済み） |

### 本部向けダッシュボード

| 項目 | 内容 |
|------|------|
| 店舗別の売上ランキング | 人件費率・目標達成率など |
| 店舗からの申請・報告 | 件数（設備故障、クレーム、キャンペーン結果など） |
| スタッフ日報の要約 | 店舗別・日/週/月（※実装済み） |
| 店長・マネージャー日報の要約 | 店舗別・日/週/月（※実装済み） |

---

## 3. シフト

- 社員用：自分のシフト一覧・カレンダー（※本日のシフト表示は実装済み）
- 店長用：店舗全体シフト、シフトCSVアップロード（※実装済み）
- データは AirシフトのCSV取り込みで表示（編集はAirシフト側）

---

## 4. 日報

- 一般スタッフ：選択式＋ひとことメモで入力（※実装済み）
- 店長・マネージャー：店長日報の入力（※実装済み）
- 要約：日/週/月単位のAI要約（※実装済み）

---

## 5. お知らせ・掲示板

- 一覧・詳細・既読管理（※一覧・詳細・投稿は実装済み）
- 店舗別・対象ロールの指定、公開期間（将来拡張）

---

## 6. まだ実装していない主な候補

| 項目 | 概要 |
|------|------|
| 勤怠・打刻 | 出退勤ボタン、打刻履歴、勤怠修正申請（店長承認） |
| マニュアル・ナレッジ | カテゴリ別マニュアル一覧・詳細（PDF等） |
| 予約・売上（閲覧） | 店長・本部向け。予約一覧、売上サマリー・目標対比 |
| タスク・チェックリスト | 今日やるべきタスク一覧、完了チェック、店長によるタスク割り当て |
| プロフィール・設定 | 氏名・連絡先、パスワード変更、通知ON/OFF |
| ユーザー・店舗管理 | 本部向け。社員アカウントの作成・無効化、店舗登録 |

---

## 7. ダッシュボードに「追加する」ときのチェックリスト（案）

- [ ] 今日のシフト（※社員用は実装済み。店長・本部にも同じウィジェットあり）
- [ ] お知らせ（※社員用は実装済み。店長・本部用に同じ or 店舗フィルタ）
- [ ] 日報の状況（本日提出済みか、未提出なら入力リンク）（※社員用は「日報」ブロックで対応済み）
- [ ] 売上速報・目標進捗（店長・本部）
- [ ] 予約状況（店長・本部）
- [ ] 出勤予定者・欠勤アラート（店長）
- [ ] 未対応タスク（店長）
- [ ] 店舗別サマリー（本部）
- [ ] マニュアルの「おすすめ」や新着（全員 or 役割別）
- [ ] タスク・チェックリストの今日分（社員・店長）

※ 実装済み＝現在のポータルにすでに入っている機能です。

この一覧を見ながら、「ダッシュボードに載せたい項目」を選んでいただければ、優先順位や画面イメージも一緒に詰められます。
